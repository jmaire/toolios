## Requirements

- iOS 11.0+
- Swift 4.2+

## Installation

toolios is available through [CocoaPods](https://cocoapods.org).

To install it in your Xcode project, specify it in your `Podfile`:

```
pod 'toolios'
```

Then, run the following command:

```bash
$ pod install
```
